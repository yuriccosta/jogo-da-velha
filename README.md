# Jogo-da-Velha

Jogo da velha feito em python  
Tem duas versões: solo (você contra o computador) e local (você contra um amigo)  
Digite a posição de cada jogada, ela vai do 1 até o 9, da esquerda para direita  
Exemplo:  
( 1 ) ( 2 ) ( 3 )  
( 4 ) ( 5 ) ( 6 )  
( 7 ) ( 8 ) ( 9 )
