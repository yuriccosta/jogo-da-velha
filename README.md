# Jogo-da-Velha
Jogo da velha feito em python <br>
Tem duas versões, solo (você contra o computador) e local (você contra um amigo local) <br>
Digite a posição de cada jogada, ela vai do 0 até o 8, da esquerda para direita<br>
Exemplo:<br>
( 0 ) ( 1 ) ( 2 )<br>
( 3 ) ( 4 ) ( 5 )<br>
( 6 ) ( 7 ) ( 8 )
